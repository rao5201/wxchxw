# 武穴茶海虾王电子商务中心 - 官方网站

## 项目简介
本项目是武穴茶海虾王电子商务中心的官方展示网站，包含核心业务介绍、AI 智能服务展示及资源对接表单。

## 安全特性
- ✅ 内容安全策略 (CSP) 防护
- ✅ 子资源完整性 (SRI) 校验
- ✅ 前端输入清洗与验证码机制
- ✅ 移动端响应式设计

## 文件结构
- `index.html`: 主页面
- `logo.jpeg`: 品牌标识

## 部署说明
直接部署到 Nginx/Apache 静态服务器即可。需配合后端接口处理表单提交。
